# Project Details

## Purpose

This is a front-end application intended to make use of the telemetry data stored in the Vespa App. Data can be importd and visualized.

## Main App functionality

- Upload of export file from Vespa App
- Processing of data
  - Vespa details
  - Route import
  - Route visualization on a map
  - Aggregation of route data on a per day basis
  - Export to GPX possibility for every route / day
  - Telemetry data

## Tech Stack

This project uses:

- React / Javascript
- Vite framework
- SQLite database
